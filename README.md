# gem-whois

Show information about the owner of a gem.

## Installation

     $ gem install gem-whois

## Usage

    $ gem whois gem-whois

    gem-whois (0.2)

    Show information about the owner of a gem

    author(s) : David Dollar
    downloads : 0
    homepage  : http://github.com/ddollar/gem-whois
